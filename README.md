# Adamant GitHub Project

Этот проект собран из загруженного файла с кодом сайта.

## Что внутри
- `index.html` — навигационная стартовая страница
- `pages/` — 10 отдельных HTML-экранов
- `.gitignore` — базовый игнор для системных файлов

## Как загрузить на GitHub
1. Создай новый репозиторий на GitHub.
2. Загрузите содержимое этой папки в репозиторий.
3. Для публикации сайта включи **GitHub Pages**:
   - Settings → Pages
   - Branch: `main`
   - Folder: `/ (root)`

## Важно
Сейчас это набор **отдельных HTML-страниц/экранов**, а не полностью связанный сайт:
- между страницами почти нет внутренней навигации
- многие кнопки пока без рабочих ссылок
- стили подключаются через CDN Tailwind и Google Fonts
